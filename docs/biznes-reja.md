# IjaraSafe — Biznes Reja
### Xavfsiz ijara platformasi · O'zbekiston

---

## 1. Muammo

O'zbekistonda ijara bozorida ishonch muammosi jiddiy:

| Muammo | Ta'siri |
|--------|---------|
| Ijarachilar pul yuborib, aldanib qoladi | Yiliga minglab firibgarlik holatlari |
| Maklerlar standart tarifga ega emas | 30-50% gacha norasmiy to'lovlar |
| Uy egalari oldindan to'lovni kafolatlay olmaydi | E'lonlar band bo'lib qoladi |
| Shaffoflik yo'q | Ijarachi narxni bilmaydi, ortiqcha to'laydi |

**Hozirgi bozor holati:** OLX, Telegram guruhlar va og'zaki — bu kanallarning hech birida to'lov himoyasi yo'q.

---

## 2. Yechim — IjaraSafe

**Escrow (garov) tizimiga asoslangan xavfsiz ijara platformasi.**

Pul to'g'ridan-to'g'ri uy egasiga emas, platformada xavfsiz saqlanadi. Ijarachi ko'chib kirganini tasdiqlaganidan keyingina uy egasiga o'tkaziladi.

```
Ijarachi to'lov qiladi → Platforma saqlaydi → Uy egasi tasdiqlaydi → Ko'chib kirish → Mablag' chiqariladi
                                                                           ↓
                                                                    Muammo bo'lsa → Admin hal qiladi
```

### Asosiy ustunliklar

| Kimga | Foyda |
|-------|-------|
| **Ijarachiga** | Pul xavfsiz, aldanmaslik kafolati |
| **Uy egasiga** | Tasdiqlangan ijarachi, kafolatlangan to'lov |
| **Maklerga** | Rasmiy komissiya (max 20%), reyting tizimi |
| **Platformaga** | Har bir bitimdan 3% daromad |
